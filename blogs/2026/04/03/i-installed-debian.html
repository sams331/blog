Life I Installed Debian April 3, 2026 | sams331

Uh ok so in this blog I will not talk about the things before, I just wanna tell that I installed Debian on the device I'm working my blog on.

I actually installed Linux Mint before I installed Debian. It was because I wanna reuse this thing so it can run things without being a security risk. (because this thing used to use Windows 7) Before installing Mint, I installed Windows 10, and because this device is... well, old, as you might expect it runs so bad that I thought of reverting to Windows 7. But I remembered there is this thing called Linux that is lightweight in some of its distros. So, I asked ChatGPT which Linux distro to install, and it reccommended Linux Mint Xfce. I installed it, and I used it 'till now. And that's the start of me jumping into the Linux world.

After using Linux Mint for a few months roughly one and a half month or something, I decided to "advance" (you advance linux users will be laughing at me for calling debian "advance" but picture this. i just switched from windows to linux this year and it hasn't been half a year for me using it. so im a literal noob in this kind of system so no hate or anything. i called debian more advance because basically it's just a little bit more manual than linux mint.). Since my HDD is gigantic (500 GB for me is gigantic) and it still has a ton of storage left, I decided not to change my OS, but doing something called dual-boot. Basically it's just having 2 OSes installed into your system that you can switch between.

After installing, it literally made my laptop more usable. It's light both in size and in resource usage (cos i installed xfce and lxqt), making my low ahh spec not cry itself. And SOUND WORKSSSSSSSSSSSSSSS!!!!!!!!!! (for some reason sound wouldn't work in mint but in debian it does) So, I can play videos and/or play my favorite music (and the music remnants from the windows 7 era). So, knowing that sound works, the thing I did that a person like me will do is install Spotify to play my playlist. (can't not use the already working sound)

If you have an old device u wanna revive if it's from like 201x or smt I recommend installing Linux Mint to start to jump in the Linux world. Want a guide to install it? Guess what? There is a sh*t ton of tutorials on Youtube. (the installer itself is pretty understandable without tutorials anyway)
